Host Setup

  1. On the host machine, start the backend with npm run dev inside backend.
  2. Start the frontend with npm run start:lan inside frontend.
  3. Open the host machine address shown by Angular, then choose Host LAN Game.
  4. Create the lobby, then share both the host URL and the six-character lobby code with every player.

Player Join

  1. Connect to the same network as the host.
  2. Open the host's frontend URL in a browser.
  3. Select Join LAN Game.
  4. Enter your display name and the game code from the host.

What to expect

  • LAN games are human-only. Bots remain available in Single Device mode.
  • Your own hand stays visible to you, but the other player's cards stay hidden.
  • If hosting fails, the backend is usually not running yet or the frontend was started without LAN mode.
  • If joining fails, re-check the host URL, game code, and that the host machine firewall allows the chosen ports.
